BTV-GLUE

This is the project page for BTV-GLUE, a GLUE project for bluetongue virus (BTV).

GLUE

GLUE is a software package for developing nucleotide-level hypotheses about the evolution of virus genomes. It enables the dissemination of these hypotheses on the web and their utilisation in wider software systems.

Included in the BTV-GLUE project

A database of BTV isolates was created using the reference collection at IAH Pirbright. Virus isolates with unique reference collection numbers are linked to genetic data. Reference isolate data includes:

  • Virus species & serotype
  • Identification method
  • Place of original sample
  • Date original sample was taken
  • Species that the isolate was taken from (e.g cow, sheep, deer etc)
  • Sample and storage information (where, by whom, sample material, passage history of sample).

This resource can be used to help identify the strain and origins of novel bluetongue viruses, and can continue to grow in value as the volume of BTV sequence increases in coming years.
